Different ways of calculating pi¶

While most of us first learned of pi as the circumfrence of a circle to its diameter, that's not the only way to find this constant. This notebook takes a look at some different ways of calculating pi and compares their usefullness, meassured in their convergence rate.

Calculating pi from random floats (monte carlo method)¶

  • Generate a list of sets of two random float numbers in the range [-1, 1], so they make up a square of sidelength two.
  • Within this square you can find the unit circle with radius 1
\begin{equation} \begin{split} \text{We know the area of a circle is given by: } A = \pi r^2 \end{split} \end{equation}
  • As the unit circle has a radius of one, we know it must have an area of π and the square must have an area of 2 * 2 = 4
  • Using the formula below
\begin{equation} \begin{split} \frac{\text{dots within circle}}{\text{total dots}} = \frac{\text{area of circle}}{\text{area of square}} \implies \frac{\text{dots within circle} \cdot \text{area of square}}{\text{total dots}} = \text{area of circle} \end{split} \end{equation}\begin{equation} \begin{split} \frac{\text{area of circle}}{\text{area of square}} = \frac{\pi r^2}{4r^2} = \frac{\pi}{4} \end{split} \end{equation}

Leibniz series for calculating pi¶

This leibniz series is a great way to calculate pi.
\begin{equation} \frac{\pi}{4}=\sum_{i=0}^N \frac{ (-1)^i }{2i+1} \end{equation}

Chudnovsky algorithm¶

The chudnovsky algorithm was first developed by the Cudnovsky brothers in the 1980's and used to break multiple world records. \begin{equation} \pi = \frac{426880 \sqrt{10005}}{\sum_{k=0}^{\infty}\frac{(6k)!(545140134 \cdot k + 13591409)}{(3k)!(k!)^{3}(-262537412640768000)^{k}}} \end{equation} As you can see from the below computation, this algorithm seems to converge quicker than python has the floating point accuracy to represent. In fact, After only two iterations, our float representation of pi is equal to numpy's pi constant. For more information on this I would suggest looking up implementations of chudnovsky.

def piCircle(n):
    x_s = []
    x_c = []
    y_s = []
    y_c = []
    for i in range(n):
        a = uniform(-1,1)
        b = uniform(-1,1)

        #Checks if the absolute value (length) of the vector is larger than one. If so it's outside the circle
        if (a**2 + b**2 ) > 1:
            x_s.append(a)
            y_s.append(b)
        else:
            x_c.append(a)
            y_c.append(b)
        
    return (len(x_c)) / (len(x_s) + len(x_c)) * 4

def leibnizPi(n):
    sum = 0
    for i in range(n):
        if (i%2 == 0):
            sum += 1/(2*i + 1)
        else:
            sum += (-1)/(2*i+1)
    return sum*4
